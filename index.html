Workflow

How Agentflow works

From issue to merged PR in minutes. No friction, no tab-switching, no context loss.

01 / CONNECT

Pull your issues

Import issues from GitHub, Linear, Jira, GitLab, or Asana directly into Agentflow. Each issue becomes a task with full context attached.

02 / ASSIGN

Choose your agent

Pick any coding agent — Claude Code, Codex, Cursor, or another. Agentflow spins up an isolated Git worktree and passes the full context.

03 / PARALLELIZE

Run agents in parallel

Launch multiple agents simultaneously. Each runs in its own worktree. Monitor them all from a single dashboard in real time.

04 / SHIP

Review, commit & push

Built-in diff view lets you review changes inline. Commit and push without leaving Agentflow. Create PRs with one click.

FAQ

Frequently asked questions

What is Agentflow? +
Agentflow is an open-source agentic development environment — your coding agent dashboard. It lets you orchestrate multiple coding agents (Claude Code, Codex, Cursor, Gemini, and 20+ more) in parallel, each isolated in their own Git worktree. You write code by directing agents, reviewing their diffs, and merging — all from a single unified interface.
Is it free? +
Yes. Agentflow is completely free and open source. You can download it today on macOS, Windows, or Linux at no cost. The source code is publicly available on GitHub.
Is my code private? +
Yes. Agentflow runs locally on your machine. Your code never leaves your environment unless you explicitly push it. When using remote SSH or cloud infra, data stays within your own provisioned workspace.
Which agents are supported? +
Agentflow supports 25+ coding agents including Claude Code, Codex, Cursor, Amp, Gemini, GitHub Copilot, Aider, Continue, Cline, OpenHands, SWE-agent, Devin, Plandex, and many more. Installed agent CLIs are auto-detected — no setup required.
Do I need API keys? +
It depends on the agent. Some agents (like Claude Code or Codex) require your own API key from the provider. Agentflow itself doesn't require any API keys — it's the orchestration layer that sits on top of the agents you already use.
What platforms are supported? +
Agentflow supports macOS, Windows, and Linux. Desktop installers are available for all three. You can also run agents remotely over SSH on any server or cloud VM.
How do I install it? +
Click the Download button at the top of the page or in the section below. Choose the installer for your platform (macOS .dmg, Windows .exe, or Linux .AppImage). Once installed, Agentflow auto-detects your coding agents and you can start orchestrating immediately.
